# Sorting Algorithm Comparison

## Project Overview

This project compares the performance of three sorting algorithms:

* Bubble Sort
* Merge Sort
* Quick Sort

The application allows users to enter a sequence of numbers and compares the execution time of each algorithm using Python.

## Features

* Compare Bubble Sort, Merge Sort, and Quick Sort
* Measure execution time using Python's time module
* Display theoretical time complexities
* Identify the fastest algorithm
* Visualize execution times using a bar chart
* Input validation for user entries
* Responsive frontend using HTML, CSS, and JavaScript

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript
* Chart.js

### Backend

* Python
* Flask

## Time Complexities

| Algorithm   | Best Case  | Average Case | Worst Case |
| ----------- | ---------- | ------------ | ---------- |
| Bubble Sort | O(n)       | O(n²)        | O(n²)      |
| Merge Sort  | O(n log n) | O(n log n)   | O(n log n) |
| Quick Sort  | O(n log n) | O(n log n)   | O(n²)      |
